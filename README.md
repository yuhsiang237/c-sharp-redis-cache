# c-sharp-redis-cache
The c-sharp-redis-cache project is a cache-service implement of id list.

<table>
<tr>
<td>Method</td>
<td>API</td>
<td>Description</td>

</tr>
<tr>

<td>Get Id list by Id.</td>
</tr>
<tr>
<td>GET</td>
<td>/Id/Refresh</td>
<td>Refresh Id list cache</td>
</tr>
</table>
